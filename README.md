# Machine Learning Projects

A collection of 11 Machine Learning projects covering all major
ML algorithms applied on real-world datasets using Python.
All projects completed during Data Analyst Internship at
Cube AI Solutions Tech Pvt Ltd.

## Projects

### 1. Salary Prediction
- Algorithm: Simple Linear Regression
- Predicted salary based on years of experience
- Tools: Python, Pandas, Scikit-learn, Matplotlib

### 2. Student Performance Prediction
- Algorithm: Multiple Linear Regression
- Predicted student scores based on multiple input features
- Tools: Python, Pandas, Scikit-learn, Matplotlib

### 3. Diabetes Prediction
- Algorithm: Logistic Regression
- Classified patients as diabetic or non-diabetic
- Tools: Python, Pandas, Scikit-learn, Seaborn

### 4. Online Fraud Detection
- Algorithm: Decision Tree
- Detected fraudulent online transactions from patterns
- Tools: Python, Pandas, Scikit-learn, Matplotlib

### 5. California Housing Price Prediction
- Algorithm: Random Forest
- Predicted housing prices based on location and features
- Tools: Python, Pandas, Scikit-learn, Seaborn

### 6. Cancer Detection
- Algorithm: K-Nearest Neighbors (KNN)
- Classified tumors as malignant or benign
- Tools: Python, Pandas, Scikit-learn, Matplotlib

### 7. Spam Detection
- Algorithm: Support Vector Machine (SVM)
- Classified messages as spam or not spam
- Tools: Python, Pandas, Scikit-learn, NLTK

### 8. Iris Flower Classification
- Algorithm: Naive Bayes
- Classified iris flowers into 3 species
- Tools: Python, Pandas, Scikit-learn, Seaborn

### 9. Medical Insurance Cost Prediction
- Algorithm: XGBoost
- Predicted medical insurance charges based on patient data
- Tools: Python, Pandas, XGBoost, Scikit-learn

### 10. Graduate Admission Prediction
- Algorithm: Gradient Boosting
- Predicted chances of graduate admission based on GRE,
  TOEFL and other factors
- Tools: Python, Pandas, Scikit-learn, Matplotlib

### 11. Credit Card Customer Segmentation
- Algorithm: Clustering
- Segmented credit card customers based on spending behavior
- Tools: Python, Pandas, Scikit-learn, Matplotlib, Seaborn

## Common Steps in Each Project
- Loaded and explored the dataset
- Performed data cleaning and preprocessing
- Split data into training and testing sets
- Applied machine learning algorithm
- Evaluated model using accuracy, precision, recall metrics
- Visualized results and derived insights
